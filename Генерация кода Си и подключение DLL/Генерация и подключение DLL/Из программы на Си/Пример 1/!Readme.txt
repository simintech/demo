В файле "Шаблон.prt" содержится шаблон для генерации библиотеки. 
Для успешной генерации библиотеки в системе должен иметься компилятор, 
выбранный в настройках компилятора.
Загрузка сгенерированной библиотеки происходит в файе "Загрузчик.prt"

Проект "Загрузчик.prt" запускает только в 32-битной версии SimInTech.

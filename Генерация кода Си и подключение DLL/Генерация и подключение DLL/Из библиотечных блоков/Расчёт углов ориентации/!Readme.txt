В файле "шаблон.prt" содержится шаблон для генерации библиотеки. Для успешной генерации библиотеки в системе должен иметься соответствующий компилятор, выбранный в настройках компилятора.
Загрузка сгенерированной библиотеки происходит в файле "загрузчик.prt"

Проект "Загрузчик.prt" запускает только в 32-битной версии SimInTech.
